### Living document of useful (to me) vim notes  

## NerdTree

Open a menu

- `:NERDTree`

When NerdTree is open

- click on folder/file to open
- `m` to open nerdtree-menu
- `a` to add a new file

## Plugins
Vim is shit without plugins  
Vim is golden with plugins

## Other vim commands

```
:w - save
:x - save and close
u - undo
v - select
:vs - vertical split
```
